# Pfand Flip! 🍾

Ein lustiges Arcade-Spiel, bei dem du Pfandflaschen in die Luft schnippst, um Punkte zu sammeln! Halte die Flasche so lange wie möglich in der Luft, erhöhe deine Combo und sichere dir den ultimativen Pfandbon im deutschen Supermarkt!

## 🎮 Spielanleitung
1. **Flasche antippen/klicken:** Halte die Flasche in der Luft, indem du sie rechtzeitig anklickst.
2. **Combos aufbauen:** Je öfter du triffst, desto höher wird der Multiplikator (bis zu 10x).
3. **Nicht fallen lassen:** Du hast 3 Leben. Wenn eine Flasche den Boden berührt, verlierst du ein Leben und deine Combo wird zurückgesetzt.
4. **Pfandbon erhalten:** Wenn das Spiel vorbei ist, wird deine Leistung berechnet und du erhältst einen ausdruckbaren (virtuellen) Pfandbon mit deinen Statistiken!

## 🚀 Features
- 🎵 **Web Audio API Soundeffekte:** Authentisches Feedback für Treffer, Combos und Spielende.
- ✨ **Partikeleffekte & Bildschirmwackeln:** Dynamisches visuelles Feedback für jeden Klick.
- 🏪 **Pfandautomat-Animation:** Realistisches Supermarkt-Feeling beim Game Over.
- 📱 **Responsive Design:** Spielbar auf Desktop und mobilen Geräten.

## 🛠️ Technologien
- HTML5 Canvas
- Vanilla JavaScript
- CSS3 (Animationen & Styling)
- Web Audio API
